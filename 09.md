# switch文
当面はif文でよい。詳しくは以下のサイトを参照のこと。
http://ufcpp.net/study/csharp/st_branch.html#switch

以下、switch文の書き方。

```cs
switch(変数){
//条件1
case 条件1:
//変数が条件1の時の処理
break; //C#では必須
case 条件2:
//変数が条件2の時の処理
break;
//：必要なだけ、caseは書ける
//これまでの条件のいずれにも該当しないとき
default:
//その他の処理
break;
}
```

---

## if文との違い
-条件判断に使うのは、特定の1つの変数
-条件が多いときに見やすくなる(ifで分岐させるのは、一般的に3つ程度)
